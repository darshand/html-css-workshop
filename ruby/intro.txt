- Ruby is a developer friendly scripting language.
- Complete Object Oriented.
- Easy to learn, its just like english.

- INSTALLATION
  - using RVM to install Ruby and Rails
  - Gemsetes

- Ruby files end with .rb extension. i,e example.rb
- To run a ruby file use 'ruby example.rb'

- No need to specify data type, ruby does it for you!
  - write numbers directly, they are considered as integers, write       number with decimal, it will recognise them as float. Write a        value within quote, they are considered as strings.
- No semicolon at the end of line!

- single line comment starts with #
- mutltiline comment starts with any of the symbol!!
  =begin
   anything goes inside is considered as comments
  =end
